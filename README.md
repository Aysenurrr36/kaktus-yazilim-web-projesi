# Kaktüs Yazılım - Staj Projesi

Bu proje, Kaktüs Yazılım için modern, SaaS (Hizmet olarak yazılım) standartlarına uygun bir kurumsal web sitesi tasarımı staj projesidir. Proje geliştirilirken yapay zeka (AI) destekli kodlama ve tasarım süreçlerinden aktif olarak yararlanılmış, temiz bir mimari hedeflenmiştir.

## 🚀 Özellikler

- **Modern Tasarım Sistemi**: "Glassmorphism" ve canlı vurgu renkleri (accent colors) ile oluşturulmuş estetik ve yenilikçi kullanıcı arayüzü.
- **Kaktiva AI Asistanı**: Kullanıcılarla etkileşime geçen, hızlı sorulara yanıt veren ve WhatsApp/Telefon iletişimine doğrudan yönlendirme yapan akıllı asistan.
- **Tamamen Özgün Kodlama**: Herhangi bir hazır CSS framework'ü (Bootstrap, Tailwind vb.) kullanılmadan, tamamen projeye özel, sıfırdan CSS ve JS ile geliştirilmiştir.
- **Responsive (Duyarlı) Tasarım**: Mobil uyumlu grid ve flexbox yapılarıyla tüm cihazlarda sorunsuz görünüm.
- **Performans ve Animasyonlar**: Akıcı mikro animasyonlar ve optimize edilmiş yükleme ekranları.
- **Başarı Hikayeleri ve Referanslar**: Veri odaklı (% büyüme, aktif kullanıcı vb.) başarı hikayesi kartları ve referans gösterimleri.

## 🛠️ Kullanılan Teknolojiler

- **HTML5** (Semantik Yapı)
- **CSS3** (Vanilla CSS, CSS Variables, Flexbox, Grid, Custom Keyframe Animations)
- **Vanilla JavaScript** (Event listeners, sessionStorage yönetimi, DOM manipulation)

## 📁 Proje Yapısı

- `index.html`: Web sitesinin ana sayfası.
- `intro.html`: Sitenin açılış (intro) sayfası.
- `style.css`: Ana sayfa için temel stil dosyası.
- `intro.css`: Açılış sayfası için stil dosyası.
- `script.js`: Tüm etkileşimler ve Kaktiva Asistan mantığının bulunduğu JavaScript dosyası.
- `AI_Prompt_Kullanimi.txt`: Proje geliştirilirken kullanılan yapay zeka komutlarının (prompt) ve izlenen yaklaşımın raporu.

## ⚙️ Kurulum ve Kullanım

Projede herhangi bir sunucu veya bağımlılık paketi (NPM vb.) bulunmamaktadır. Projeyi çalıştırmak için:

1. Proje klasörünü indirin veya klonlayın.
2. Klasör içerisindeki `intro.html` veya doğrudan `index.html` dosyasına çift tıklayarak modern bir web tarayıcısında (Chrome, Firefox, Safari, Edge vb.) açın.

## 📝 Notlar

Proje sürecinde karşılaşılan mobil uyumluluk, asistan davranışı ve performans gibi problemler, yapay zeka yardımıyla algoritmik olarak çözülmüş ve kod kalitesi artırılmıştır. Detaylı bilgi için `AI_Prompt_Kullanimi.txt` dosyasını inceleyebilirsiniz.
